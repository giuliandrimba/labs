# Setup

````bash
git submodule init
git submodule update
cd vendors/theoricus
npm install
cd ../../
````

# Starting app

````bash
make s
````

Then visit [http://localhost:11235](http://localhost:11235)

# Releasing app

````bash
make r
````

# Help

For more options, run:

````bash
make h
````
